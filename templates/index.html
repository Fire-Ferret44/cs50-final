Welcome to the Call Roster Generator

This is a webapp dedicated to roster shift calendars, timetables or similar problems that have multiple participants and constraints. It allows you to create a roster for your team, taking into account various constraints and preferences. The roster can be generated based on the input you provide, and you can download it in various formats.

It might not be initially obvious how to interact with the site so please go to the tutorial page first.

The site is built in a modular way and parameters can be adjusted, but the current "standard" model used as an example is for a working hours plus overtime call schedule and is as follows:

Monday - Thursday: 1x 16h00-20h00, 1x 16h00-10h00+
Friday: 2x 16h00-08h00+
Saturday: 2x 08h00-08h00+
Sunday: 2x 08h00-10h00+

If you have any questions or suggestions please use the contact link above. Good luck and enjoy!
